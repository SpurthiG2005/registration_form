# Online Registration Form (HTML + CSS + JS + jQuery + PHP)

## Objective
A simple web application that collects user information through an online form, validates input using JavaScript/jQuery, processes it using PHP, and displays formatted output.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- jQuery
- PHP
- XAMPP (Local server)
- GitHub (Hosting)

## How to Run (Local)
1. Install XAMPP.
2. Copy the project folder into:  
   `C:/xampp/htdocs/registration_project`
3. Start **Apache** in XAMPP control panel.
4. Open browser and visit:  
   `http://localhost/registration_project/index.html`

## Hosting on GitHub
GitHub does not run PHP.  
So only frontend (HTML/CSS/JS) will be visible.

Upload the entire project to your GitHub repository.

Your PHP processing will work only on XAMPP or a PHP-supported server.
